# daozang-english
This repository contains a mostly complete english translation of the Zhengtong Daozang, the orthodox daoist canon.

#### About This Translation
This repository contains English translations of approximately 1,700 texts
from the Daoist Canon (道藏 Daozang), created using AI-assisted translation
using semantic- based, punctuation- based and marker- based chunking. 

**Translation Status:**
- Source: Chinese texts from [original GitHub repos - cite them!]
- Method: AI translation (Mistral Large 2512 + Mistral 14b)
- Chunking: 256 characters with contextual overlap and the complete previously translated chunk of the text
- Quality: First-pass translation requiring scholarly review
